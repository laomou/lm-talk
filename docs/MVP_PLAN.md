# LM Talk MVP Plan

Current MVP tracks:

1. Identity create/restore and encrypted backups.
2. Contact Cards, friend requests/responses, and local blocking.
3. X3DH PreKey bundle publishing/getting and Double Ratchet session state.
4. Direct chat through WebRTC, Mailbox, or outbox retries.
5. Group fanout and Sender Key experimental path.
6. File packages with manual download after receipt.
7. PWA/offline shell and IndexedDB encrypted local state.

Remaining long-term work: production DHT replication, relay/TURN strategy, MLS or full group protocol, stronger backup UX, extensive property tests and audit.
