# LM Talk MVP 计划

当前 MVP 轨道：

1. 身份创建/恢复和加密备份。
2. 联系人名片、好友请求/响应和本地拉黑。
3. X3DH PreKey bundle 发布/获取和 Double Ratchet 会话状态。
4. 通过 WebRTC、Mailbox 或 outbox 重试进行私聊。
5. 群组 fanout 和 Sender Key 实验路径。
6. 文件包收取后手动下载。
7. PWA/offline shell 和 IndexedDB 加密本地状态。

剩余长期工作：生产级 DHT replication、relay/TURN 策略、MLS 或完整群协议、更强的备份 UX、fuzz/压力测试和安全审计。
